# UI-UX-Experiment2
## Aim:

To Design a Basic Mobile and Web Layout for an Existing Case Study – “TeenNova” T-Shirt Shopping Application.

## Algorithm:
1. Study the requirements of a modern fashion e-commerce application.
2. Apply Gestalt principles to improve visual hierarchy and user experience.
3. Design mobile and web layouts using Canva.
4. Add UI elements such as navigation bars, product cards, buttons, icons, and forms.
5. Develop interaction behaviors and usability principles for smooth user interaction.
6. Create a stylish brand logo for “TeenNova” following branding guidelines.
7. Export the final layouts and organize them into a GitHub repository.

## procedure:
Design a Basic Mobile and Web Layout for an Existing Case Study
For “TeenNova,” Gestalt principles improve the shopping experience and visual appeal:

### Proximity:

Related items such as product image, price, ratings, and “Add to Cart” buttons are grouped together to reduce confusion and improve readability.

### Similarity:

Consistent orange and beige color palettes, rounded buttons, typography, and card styles create a harmonious and modern fashion interface across both mobile and web layouts.

### Figure-Ground:

A soft light background with bold orange UI sections creates strong contrast, helping important content such as featured products, offers, and CTA buttons stand out clearly.

### Continuity & Closure:

Aligned product sections, smooth spacing, and rounded card layouts guide the user naturally through the interface while maintaining a clean and minimal shopping experience.

By applying these principles, the TeenNova app provides an attractive, organized, and user-friendly fashion shopping platform.

#### The TeenNova logo was designed in Canva using:

A minimalist T-shirt illustration
Modern rounded typography
Orange and black color combinations
Fashion-inspired branding elements

The logo represents a trendy and youthful streetwear fashion brand.
Its simple vector-based structure makes it suitable for:

Mobile app icons
Website headers
Splash screens
Marketing banners

The logo maintains strong visual clarity and brand consistency across both web and mobile platforms.
